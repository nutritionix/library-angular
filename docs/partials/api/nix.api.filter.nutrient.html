nutrient
filter in module nix.api

Description

Finds nutrition with specified id in array of nutrients

Usage

In HTML Template Binding

{{ nutrient_expression | nutrient:id:[attribute] }}

In JavaScript

$filter('nutrient')(nutrients, id, [attribute])

Parameters

ParamTypeDetails
nutrientsarray

Collection of nutrients

idstring

Nutrient attr_id to search for

[attribute]string

property to immediately return from found nutrient

Returns

mixed

Nutrient object itself, or it's property named with the third param
